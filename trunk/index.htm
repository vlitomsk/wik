Главная

Вашему вниманию представлена игра, посвященная истории Омского театра юного зрителя и спектаклю Путешествие профессора Тарантоги. Это компьютерный вариант творческой игры, проводимой городским клубом интеллектуальных игр Мгновение.

Во все времена театр воспитывал и просвещал, давал пищу для ума и способствовал проявлению лучших человеческих качеств. Старая истина — все мы родом из детства, в том числе и из театрального детства. Никогда не будет забыт тот постулат, что спектакли для детей надо ставить так же, как для взрослых, только лучше.

Наш ТЮЗ всегда радует своих юных зрителей новыми яркими постановками, развивает их вкус, формирует способность чутко воспринимать прекрасное, доброе, вечное. Умные, добрые, содержательные спектакли неизменно оставляют глубокий след в детских душах.

Несколько поколений омичей считают ТЮЗ своим первым театром, здесь произошла для них встреча со сценическим искусством. Теперь повзрослевшие бывшие юные зрители приводят сюда своих детей, внуков.

Этапы игры

  • на первом этапе необходимо вводить ответы на задания в поле ввода,
  • на втором этапе необходимо сортировать какие-то объекты по какому-то признаку.

Подготовка к игре

  • познакомиться с краткой историей Омского ТЮЗа (раздел История),
  • посмотреть спектакль Путешествие профессора Тарантоги.

Из любого места игры можно перейти на Начало щелчком по соответствующей кнопке.
